# zenzei_os

Zenzei OS is a sleek, lightweight, browser-based operating system simulation inspired by futuristic themed space interfaces.
(^-^)
It is built entirely with:
#HTML5 
#CSS3 
#JavaScript

## Features

* **Desktop Environment**: You get a custom window manager, a starfield background and special sound effects.

* ** Starfield**: The starfield looks super realistic with moving stars and a cool glow.

* **Glassmorphic Design**: The windows and menus look like glass and have a nice glow to them.

## What You Can Do

* **Window Manager**: You can open applications at once move windows around and resize them.

* **Taskbar & Start Menu**: There's a start menu that shows the time, battery life and system info.

* **Synthesized Audio**: The system makes its sounds for startup, clicks and notifications.

## Built-in Applications

* **Cosmic Terminal**: This is like a command line where you can type commands and see system info.

* **Nebula Notes**: A text editor that saves your work automatically.

* **Cosmic Asteroids**: A game where you shoot asteroids.

* **Stellar Draw**: A drawing app where you can make art with brushes and colors.

* **Space Explorer**: You can learn about space things like stars and planets.
